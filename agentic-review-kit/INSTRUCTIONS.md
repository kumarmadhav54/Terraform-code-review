This folder contains everything you need to enable **Agentic AI Code Reviews** in your Terraform project.

## How to Install (Plug & Play)

1.  **Copy Files**:
    *   Copy `review.sh` to the root of your project.
    *   Copy the `.vscode` folder to the root of your project (merge it if one exists).

2.  **Make Executable**:
    *   Run: `chmod +x review.sh`

3.  **Run**:
    *   In VSCode: Press **Cmd+Shift+P** -> `Tasks: Run Task` -> `Run Terraform Review`.
    *   Or Terminal: `./review.sh`

## How to Use with AI

Open `AGENTIC_SHOWCASE.md` (included in this kit) to learn how to ask RooCode (or Copilot/Cursor) to automatically evolve this script!
